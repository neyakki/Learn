    Основы HTML¶

    Ссылки

    Теги можно посмотреть:

    • w3s;
    • html book

    HTML (Hypertext Markup Language) - это язык разметки, который используется для структурирования и отображения веб-страницы и её контента. HTML состоит из ряда элементов, которые используются, чтобы вкладывать или оборачивать различные части контента, заставить контент отображаться или действовать определённым образом.

    Элементы¶

    Элемент состоит из следующих частей:

    • Открывающий тег - имя элемента, заключенного в угловые скобки. Указывает, где элемент начинается;
    • Закрывающий тег - тоже самое, что и открывающийся тег, только добавляется косую черту перед именем. Указывает, где элемент заканчивается;
    • Контент - содержимое элемента;
    • Элемент - все выше перечисленное.

    Элемент может содержать атрибуты. Они содержат дополнительную информацию об элементе, которая не будет показана в контенте.

    Атрибут всегда должен иметь:

    • Пробел между ним и именем элемента (другим атрибутом);
    • После имени атрибута следует знак равенства;
    • Значение заключено в кавычки.

    Вложенные элементы¶

    Элементы могут содержать в себе другие элементы, тем самым создавая вложенность.

    <p>Какой-то <strong>текст</strong></p>

    При этом должен соблюдаться порядок элементов. Следующий пример неверен:

    <p>Какой-то <strong>текст</p></strong>

    Пустые элементы¶

    Некоторые элементы не имеют контента, поэтому называются пустыми.

    <img src="..." alt="..." />

    Блочные и строчные элементы¶

    Существует две категории элементов:

    • Блочные - формируют видимый блок на странице - они окажутся на новой строке после любого контента, следующий за ними контент также окажется на новой строке.
    • Строчные - содержатся в блочных элементах и окружают малую часть информации.

    Примечание

    HTML5 переопределил категории элементов в HTML. Хотя эти определения точнее и однозначнее, чем те, которые были раньше, их гораздо сложнее понять, чем «блочный» и «строчный». Новое определение

    HTML документ¶

    <!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <title>Document</title>
    </head>
    <body></body>
</html>

    HTML документ состоит из:

    • <!DOCTYPE> предназначен для указания типа текущего документа;
    • html - Корневой элемент веб-страницы, все остальные элементы должны быть его потомками;
    • head - Содержит метаинформацию о странице: заголовки, скрипты, стили и т.д.;
    • meta - Представляет метаинформацию;
    • title - Устанавливает заголовок для страницы;
    • body - Содержит весь контент, который будет отображаться на странице.

    Форматирование¶

    Заголовки¶

    Заголовки создаются с помощью тегов <hn>, где n - число от 1 до 6. Заголовки должны соблюдать иерархию, т.е. первым используется заголовок h1, за ним заголовок h2 и т.д.

    <h1>Заголовок h1</h1>
<h2>Заголовок h2</h2>
<h3>Заголовок h3</h3>
<h4>Заголовок h4</h4>
<h5>Заголовок h5</h5>
<h6>Заголовок h6</h6>

    Рекомендации по использованию заголовков

    • Использовать заголовок h1 один раз на странице.
    • Использовать заголовки в правильном порядке, например, заголовок h3 не должен использоваться до заголовка h2

    Абзацы¶

    Для создания абзаца используется тег <p>

    <p>Это параграфф</p>

    Особые типы текста¶

    Элементы форматирования были предназначены для отображения особых типов текста:

    • <b> — жирный текст;
    • <strong> - важный текст;
    • <i> — текст курсивом;
    • <em> — выделенный текст;
    • <mark> — выделенный текст;
    • <small> — текст меньшего размера;
    • <del> — удаленный текст;
    • <ins> — вставленный текст;
    • <sub> — текст подстрочного индекса;
    • <sup> — текст надстрочного индекса.

    Списки¶

    Неупорядоченные¶

    Порядок элементов не важен. Для создания используется тег <ul>, тег <li> означает элемент списка.

    <ul>
    <li>молоко</li>
    <li>яйца</li>
    <li>хлеб</li>
    <li>хумус</li>
</ul>

    Упорядоченные¶

    Порядок элементов имеет значение. Создается с помощью тега <ol>.

    <ol>
  <li>молоко</li>
  <li>яйца</li>
  <li>хлеб</li>
  <li>хумус</li>
</pl>

    Список описаний¶

    Список терминов с описание каждого термина. Создается с помощью тега dl, тег dt определяет термин, тег dd определяет описание.

    <dl>
    <dt>Coffee</dt>
    <dd>- black hot drink</dd>
    <dt>Milk</dt>
    <dd>- white cold drink</dd>
</dl>

    Гиперссылки¶

    Гиперссылка - элемент контента, позволяющий перейти на другой элемент.

    Ссылка создается с помощью тега <a> и атрибута href, который содержит URL адрес или файл на локальном ПК.

    Можно ссылаться на определенную часть документа, для этого нужно указать у элемента атрибут id и в ссылке указать значение этого id.

    <h1 id="head1">Heading</h1>

<a href="index.md#head1">Head1</a>

    Цитаты¶

    Блочные цитаты¶

    <blockquote
  <p>Цитата</p>
</blockquote>

    Строчная цитата¶

    <q>Цитата</q>

    Элемент цитирования — <q> — "предназначен для коротких цитат, не требующих прерывания абзаца".

    Аббревиатуры¶

    Обеспечивает расшифровку сокращения. Будет показано при наведение на сокращение.

    <p>Здесь <abbr title="Аббревиатура">аббр</abbr></p>

    Мультимедиа¶

    Изображения¶

    Для размещения изображения на странице используется тег <img> и атрибут src.

    Атрибут alt задает текст, который будет отображаться если изображение не загрузится. А заголовок <title> показывает всплывающую подсказку при наведении.

    <img src="picture.png" alt="picture" title="picture" />

    Чтобы добавить к изображению заголовок используются теги <figure> и <figcaption>.

    <figure>
    <img src="picture.png" alt="picture" title="picture" />

    <figcaption>Picture</figcaption>
</figure>

    Видео¶

    Тег <video> позволяет добавить видео на нашу страницу.

    <video controls>
    <source src="rabbit320.mp4" type="video/mp4" />
    <source src="rabbit320.webm" type="video/webm" />
    <p>
        Ваш браузер не поддерживает HTML5 видео. Используйте
        <a href="rabbit320.webm">ссылку на видео</a> для доступа.
    </p>
</video>

    Атрибут controls добавляет возможность контролировать видео и аудио.

    Каждый тег <source> предоставляет разные форматы видео-файлов. Атрибут type предоставляет MIME type, что позволяет браузерам проверить кодек и пропустить его если, он не поддерживается.

    Параграф внутри тега <video> - резервный контент. Он будет отображаться, если браузер не поддерживает тег <video>.

    Тег <track> позволяет добавлять субтитры к видео.

    Аудио¶

    Тег <audio>работает точно так же, как тег <video>.

    <audio controls>
    <source src="viper.mp3" type="audio/mp3" />
    <source src="viper.ogg" type="audio/ogg" />
    <p>
        Your browser doesn't support HTML5 audio. Here is a
        <a href="viper.mp3">link to the audio</a> instead.
    </p>
</audio>

    Таблица¶

    Таблица - это структурированный набор данных, состоящий из строк и столбцов (табличных данных).

    В HTML для создания таблиц существует набор семантических тегов:

    • <table> - Определяет блок как таблицу;
    • <thead> - Группирует заголовок таблицы (По семантике похож на <header>);
    • <tbody> - Группирует основной блок таблицы (По семантике похож на <main>). Может использоваться несколько раз, тем самым разделяя таблицу на отдельные блоки;
    • <tfoot> - Футер таблицы (По семантике похож на <footer>);
    • <th> - Определяет заголовок столбцов таблицы;
    • <tr> - Определяет строку таблицы;
    • <td> - Определяет столбцы таблицы.
    • <caption> - Определяет подпись к таблице.

    Атрибуты colspan и rowspan позволяют объединять ячейки.

    <table>
    <caption>
        Цены на флагманские модели iPhone и Xiaomi
    </caption>
    <thead>
        <tr>
            <th>Производитель</th>
            <th>Модель</th>
            <th>Цена</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td rowspan="3">iPhone</td>
            <td>12 Pro</td>
            <td>$999</td>
        </tr>
        <tr>
            <td>12</td>
            <td>$799</td>
        </tr>
        <tr>
            <td>12 mini</td>
            <td>$699</td>
        </tr>
    </tbody>
    <tbody>
        <tr>
            <td rowspan="2">Xiaomi</td>
            <td>Mi 10</td>
            <td>$768</td>
        </tr>
        <tr>
            <td>Black Shark 3 128 Gb</td>
            <td>$529</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td colspan="2">Средняя цена:</td>
            <td>$758.8</td>
        </tr>
    </tfoot>
</table>

    Формы¶

    Форма HTML используется для сбора вводимых пользователем данных.

    Элемент HTML <form> используется для создания HTML-формы для ввода данных пользователем. Он является контейнером для других элементов ввода.

    Элемент HTML <form> может содержать один или несколько из следующих элементов формы:

    • <input> - Определяет поле ввода. Может отображаться несколькими способами, в зависимости от type атрибута:

      • button - Кнопка;
      • checkbox - Поле флажок;
      • color - Определяет блок выбора цвета;
      • date - Поле ввода даты;
      • datetime-local - Поле ввода даты и времени без часового пояса;
      • email - Поле ввода почты;
      • file - Поле выбора файла;
      • hidden - Скрытое поле ввода;
      • image - Изображение как кнопка отправки;
      • month - Поле выбора года и месяца;
      • number - Поле числового ввода;
      • password - Поле ввода пароля;
      • radio - Поле переключатель;
      • range - Элемент управления для ввода числа (например, ползунок);
      • reset - Кнопка сброса данных формы;
      • search - Поле поиска;
      • submit - Кнопка для отправки данных из формы;
      • tel - Поле ввода телефона;
      • text - Однострочное поле ввода;
      • time - Выбор времени без часового пояса;
      • url - Поле ввода URL;
      • week - Поле выбора недели;

    • <label> - Определяет метку для нескольких элементов формы;

    • <select> - Определяет раскрывающийся список. Элемент <option> определяет опцию, которую можно выбрать;
    • <textarea> - Определяет многострочное поле ввода;
    • <button> - Определяет кнопку;
    • <fieldset> - Используется для группировки связанных данных в форме;
    • <legend> - Определяет заголовок для <fieldset>;
    • <datalist> - Определяет список предопределенных параметров для <input> элемента;
    • <output> - Представляет собой результат вычисления ;
    • <optgroup> - Определяет группу связанных параметров в раскрывающемся списке.
    <form action="">
    <label for="fname">First name:</label><br />
    <input type="text" id="fname" name="fname" value="John" /><br />
    <input type="submit" value="Submit" />
</form>

    Структура веб-сайта¶

    Веб-страницы состоят из следующих элементов:

    • Заголовок;
    • Навигационное меню;
    • Основное содержимое;
    • Боковая панель;
    • Футер.

    Семантические теги — это теги, которые предназначены для того чтобы компьютерные программы (поисковые системы, сборщики информации, речевые браузеры и т.д.), понимали какой тип информации заложен в данных тегах.

    Для реализации такой структуры в HTML предусмотрены семантические теги:

    • Заголовок:
      • <header> - определяет заголовок. Может находиться в тегах <article> и <section>;
    • Навигационное меню:
      • <nav> - содержит навигацию;
    • Основное содержимое:
      • <main> - предназначен для уникального содержимого этой страницы. Используется только один раз на странице;
      • <article> - окружает блок связанного содержимого, которое несет смысл сама по себе;
      • <section> - группирует одну часть страницы, которая представляет собой одну часть функциональности;
    • Боковая панель:
      • <aside> - контент, который не имеет прямого отношения к основному содержимому, но содержит дополнительную информацию;
    • Футер:
      • <footer> - конечное содержимое страницы.
