# Авторизация и аутентификация

Обычно процедура входа разбита на несколько простейших шагов:

-   Идентификация пользователя;
-   Аутентификация пользователя;
-   Авторизация пользователя.

Идентификация - процедура, в результате которой происходит выявления идентификатора, явно определяющего пользователя в информационной системе.

Аутентификация - процедура проверки пользователя, например путем сравнения введенного пароля с паролем в БД.

Авторизация - предоставление пользователю прав на выполнение определенных действий.

Аутентификация без идентификации не имеет смысла, как можно проверять неизвестно кого.

Идентификации без аутентификации также не возможна, мало ли кто ввел логин, нужно быть уверенными что кто ввел логин действительно тот за кого себя выдает.

А авторизация возможна и без идентификации и аутентификации. Например, Google Документах можно создать документ с открытым доступом.
