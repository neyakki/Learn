# План обучения

Предпочтительное направление моего интереса охватывает две области: веб-разработку и машинное обучение. С учетом вышеописанных областей интереса, я обратился к ChatGPT с запросом на составление плана обучения, охватывающего обе указанные сферы. Подчеркиваю, что разработанный план не является окончательным, поскольку я предполагаю его постепенное уточнение и дополнение. Также в данном контексте предоставляется мини-план по изучению компьютерных наук, который был обнаружен на платформе [Github](https://github.com/ilmoi/teachyourselfCS-RU/blob/master/Teach_yourself_cs--RU.md).

## Web-разработка

### Основы веб-разработки

1. HTTP и HTTPS:

    - Зачем нужен протокол HTTP;
    - Различие между HTTP и HTTPS.

2. Основы работы с веб-серверами:

    - Как работает веб-сервер;
    - Различные веб-серверы (например, Apache, Nginx).

3. HTML и CSS:

    - Разметка и стилизация веб-страниц;
    - Семантические теги в HTML.

4. JavaScript:

    - Основы языка (переменные, циклы, функции);
    - DOM (Document Object Model) и работа с ним;
    - AJAX и Fetch API;
    - Фреймворк.

### Базы данных

1. Основы SQL:

    - Создание таблиц;
    - Выборка данных;
    - Обновление и удаление данных.

2. Работа с реляционными базами данных:

    - Знакомство с MySQL, PostgreSQL, или SQLite;
    - Нормализация баз данных;

3. Основы NoSQL:

    - Знакомство с MongoDB или другими NoSQL базами данных.

### Серверное программирование

1. Язык программирования для backend;
2. Фреймворки;
3. RESTful API;

### Расширенный frontend

1. CSS анимации и трансформации:

    - Создание анимаций с использованием CSS.

2. Canvas и SVG:

    - Работа с графикой и векторной графикой на веб-странице.

3. Работа с формами:
